# Student Management System

A C# .NET Windows Forms application for managing student records using SQLite.

## Features
- Dual Login: Admin and Student
- Add, Edit, Delete student records
- Data stored in SQLite (`studentdb.sqlite`)
- Clean, user-friendly interface
- Student dashboard (view-only)

## Setup Instructions
1. Clone or download the repo
2. Open `.sln` in Visual Studio
3. Restore NuGet packages
4. Install `System.Data.SQLite`
5. Press F5 to run

## Login Details
- Admin: `admin` / `password`
- Student: Enter any Student ID (e.g., `32211`)
